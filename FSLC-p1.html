Introduction

Free-space laser communication is a method of using lasers to transmit data without the use of media, such as optical cables. I have been interested in the properties of lasers and its applications for more than a year. When I learned that NASA is now working on developing free-space laser communication technologies, I became curious due to its inherent advantages, which are as below:

  1. Free-space laser communication does not require a fiber optic cable. This makes it adaptable to areas where installing infrastructures are a challenge, such as space communication, communication at remote areas, disastrous areas, and emergency requirements.

  2. Since free-space communication requires line of sight conditions, it is totally secured, compared to radio communication.

  3. Since laser beams are collimated and coherent, there is low beam divergence, which results in tremendous power saving. This power saving property is extremely useful in long distance space-ship communication and applications where equipment size is constrained to be small.

As I tried to design a free-space laser communication system, I could not find enough research material that was relevant to my requirement. That prompted me to start this project. This project focused on researching to find the optimum design for a free-space laser communication transceiver system. In this project, a signal was superimposed on a carrier laser beam, using amplitude modulation. The bias voltages were optimized for the system by plotting its input/output characteristics and choosing the right operating voltage range.

Purpose: Is it possible to design a free-space (wireless) laser communication system, with commonly available components, that uses laser as a carrier wave to carry audio signals with minimum distortion?

Objective

The objective of this experiment was to design a modulator circuit for a laser beam and test if the laser beam could be successfully modulated to carry an audio signal through free space.

It is possible to design a modulator for audio signals using a class A amplifier circuit made out of commonly available components. This modulator can modulate the output power of a laser source linearly. The emitted laser can be detected wirelessly at a distance and demodulated to retrieve the audio signal with minimum distortion.

  1. Using commonly used transistors, it is possible to design a class A amplifier for audio signals. (Class A amplifiers have linear input/output characteristics, which mean that the signals are amplified without distortion. Class A amplification happens over specific voltage ranges.)

  2. The output currents of the above amplifier can be used to modulate a laser source.

  3. Laser diodes, when biased forward, have power outputs that are linearly related to the current over specific operating ranges.

  4. Laser detectors, such as photovoltaic cells, have a range of output voltages that are linearly related to the incident radiation power.

Considering the above premises, it is possible that there could be an overlapping voltage range that is common to all the above systems where their input/output characteristics are linear.

Stage 3

Amplifier Linear Output Range and Overlap with Laser Voltage Range

The class A biased single stage audio amplifier, from stage 2, was configured to drive and modulate the laser. It was then given different input voltages and the output voltages were measured. The objective was to find if there was an output voltage range for the class A amplifier that would match with the input voltage range of the laser. The match should be in such a way that both these devices would be aligned within the same voltage range that has linear input/output relationship.

The base bias resistor was replaced by a trimmer potentiometer in order to facilitate variable biasing at the transistor base.

Laser Modulator

For each supply voltage, I changed the bias voltages over a range using the potentiometer, and observed the voltage across the laser for each distinct value. This was to see if it was within the laser’s linear voltage range.

Input Voltage (Laser Voltages in Linear Range): 3.4 V

Base Voltage Vs Laser Voltage - Input: 3.4V

Since VL was linearly varying with VB (base voltage), the modulator operated in class A configuration

Even though this configuration was offering a linear range between 1.54V and 1.76V for the VB, this entire range was not usable since the linear input/output voltage range for the laser did not exactly coincide with this range.Comparing the graph from Stage 2 as well as the current graph above, I got the following ranges for VL:

  1. Range of 2.447V to 2.597V from Stage 2
  2. Range of 2.42V to 2.54V from Stage 3

The overlapping range was from 2.447V to 2.54V. This was the only range where the laser output linearly varied with the base voltage of the modulator for a supply voltage of 3.4V

The voltage bandwidth = 2.54V - 2.447V = 0.093V

I repeated the above analysis with a supply voltage of 3.3 V

Input Voltage (Laser Voltages in Linear Range): 3.3 V

Base Voltage Vs Laser Voltage - Input: 3.3V

Comparing the graph from Stage 2 as well as the curent graph above, I got the followoing ranges for VL:

  1. Range of 2.447V to 2.597V from Stage 2
  2. Range of 2.46V to 2.6V from Stage 3

The voltage bandwidth = 2.597V - 2.46V = 0.137V


Therefore, the two scenarios were:

  1. For Supply Voltage = 3.4V
    Voltage Range for Linear Operation = 2.447V to 2.54V
    Therefore, Voltage Bandwidth = 2.54V – 2.447V = 0.093V

  2. For Supply Voltage = 3.3V
    Voltage Range for Linear Operation = 2.46V to 2.597V
    Therefore, Voltage Bandwidth = 2.597V – 2.46V = 0.137V

Laser Voltage Bandwidth For Supply Voltages

The supply voltage of 3.3V gave a better bandwidth. Therefore 3.3V was the ideal supply voltage.

Conclusion

Results and Findings

The following supply voltage and base voltage would give a linear input/output characteristic as well as the widest linear range for the entire system (that included the transmitter and the receiver that was designed in this project).

  • The Supply Voltage = 3.3V DC
  • The Base Bias Voltage = 1.5315V DC

This will also ensure that,

  • The Range of Potential Difference across the Laser = 2.46V to 2.597V


Transmitter/Receiver System

I was able to design a modulator for audio signals using a class A amplifier circuit that modulated the output power of a laser source linearly, so that the emitted laser beam could be detected wirelessly at a distance and demodulated to retrieve the audio signal with minimum distortion.
